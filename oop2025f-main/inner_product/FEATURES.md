# Problem Statement

Write a **function** to compute the **inner product** (or **dot product**) of two **vectors** when it is defined. For example,

      [1, 0] · [1, 1] = 1,
      [1, 1, 0] · [0, 1, 1] = 1, and
      [1, 0] · [1, 1, 0] => undefined.
